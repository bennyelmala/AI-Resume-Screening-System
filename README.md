# 📖 Overview

The AI Resume Screening System is an intelligent recruitment application designed to automate the process of screening and ranking resumes using Artificial Intelligence, Machine Learning, and Natural Language Processing (NLP).

The system analyzes candidate resumes, extracts relevant information, evaluates qualifications, and ranks applicants based on predefined hiring criteria. By automating resume evaluation, the project reduces manual screening effort, improves recruitment efficiency, and enables faster identification of qualified candidates.

This project demonstrates the practical application of AI technologies in Human Resource (HR) analytics and recruitment automation.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/bennyelmala/AI-Resume-Screening-System.git
```

## Navigate to Project

```bash
cd AI-Resume-Screening-System
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

# ▶️ Running the Application

If using Python:

```bash
python app.py
```

